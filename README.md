# TDP Cloud UI

[Cloud Panel](https://github.com/tdp-resource/cloud-panel) 的前端组件, 框架基于 [vue-admin-template](https://github.com/rehiy/vue-admin-template)

## 运行开发服务

在项目目录运行  `serve.bat` 或 `./serve.sh`

## 编译并打包

在项目目录运行  `build.bat` 或 `./build.sh`
