# 源码目录结构解析

## 下表中 `*talbe*` 表示和数据表名一一对应的文件或文件夹

```shell
TDP Cloud UI
│
├─api ------------------------------------ 接口
│  ├─alibaba ----------------------------- 阿里云接口
│  ├─basic ------------------------------- 公共服务类
│  ├─cloudflare -------------------------- Cloudflare 接口
│  │  └─typings -------------------------- Cloudflare 类型库
│  ├─native ------------------------------ 原生接口，及*table*服务类
│  └─tencent ----------------------------- 腾讯云接口
│      └─typings ------------------------- 腾讯云类库
├─apps ----------------------------------- 页面
│  ├─dashboard --------------------------- 首页
│  ├─domain ------------------------------ 域名管理
│  ├─error ------------------------------- 错误页
│  ├─layout ------------------------------ 框架
│  ├─machine ----------------------------- 主机管理
│  ├─passport ---------------------------- 用户中心
│  ├─script ------------------------------ 脚本管理
│  ├─taskline ---------------------------- 任务历史
│  ├─terminal ---------------------------- SSH 终端
│  ├─vendor ------------------------------ 供应商管理
├─assets --------------------------------- 静态文件
│  ├─img --------------------------------- 图片
│  └─style ------------------------------- 样式
├─component ------------------------------ 复用组件
├─helper --------------------------------- 辅助类
│  └─script ------------------------------ 常用脚本
├─provider ------------------------------- 厂商集成组件
│  ├─alibaba ----------------------------- 阿里云
│  ├─cloudflare -------------------------- Cloudflare
│  ├─tencent ----------------------------- 腾讯云
│  │  ├─cvm ------------------------------ CVM
│  │  ├─dnspod --------------------------- DnsPod
│  │  └─lighthouse ----------------------- 轻量服务器
│  └─worker ------------------------------ 子节点管理
├─router --------------------------------- 路由治理
└─store ---------------------------------- 状态管理
```
